# Recording

First feature beyond the updater. Captures audio from the default microphone
into a WAV file in app private storage; displays the active mic name.

## Scope

Included:
- Start/stop recording button on main screen.
- Display of currently active mic name.
- Display of elapsed recording time.
- WAV file written to `filesDir/recordings/bird_<timestamp>.wav`.
- Display of last saved file name and size.

Not included (intentional):
- F-tag dev overlays (`tagF1`, `tagF7`, `tagF8` from legacy). Tagging system
  retired; if a debug overlay returns later, it will use memory IDs (n201,
  n202, ...), not the F-scheme.
- Overflow menu, delete-all action — deferred. May return as separate nodes.
- dBFS level meter (legacy F4) — was dropped on legacy main; not reintroduced.

## Source of code

Recording loop, WAV header, mic name come from the `legacy` branch:
- `app/src/main/kotlin/com/iamweasel89/birdscope/MainActivity.kt`:
  functions `startRecording`, `writePcmToWav`, `buildWavHeader`,
  `writeIntLE`, `writeShortLE`, `stopRecording`, `formatElapsed`,
  `micDisplayName`, `requestMicAndStart`, `micPermissionLauncher`.
- `app/src/main/AndroidManifest.xml`: `RECORD_AUDIO` permission.
