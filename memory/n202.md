# Spectrum + dBFS stats

Adds live frequency-domain visualization and signal statistics to the
recording pipeline.

## Scope

Included:
- Live FFT spectrum view, updated during recording.
- Max and min dBFS values, computed during recording.
- Stats reset when recording starts.

Not included:
- Persistence of spectra across recordings.
- Configurable FFT parameters (window size, hop) — uses legacy defaults.
- Spectrogram (time axis); only current spectrum is shown.

## Source of code

From the `legacy` branch:
- `app/src/main/kotlin/com/iamweasel89/birdscope/FftAnalyzer.kt` —
  transferable unchanged.
- `app/src/main/kotlin/com/iamweasel89/birdscope/SpectrumView.kt` —
  transferable unchanged.
- Stats logic (`maxPeakSample`, `minPeakSample`, `updateStats`,
  `resetStats`) from `legacy/MainActivity.kt`.

## Depends on

n201 (Recording) — spectrum and stats are computed during recording,
so the recording pipeline must be present first.

## Status

Plan only. Build after n201 is done.
