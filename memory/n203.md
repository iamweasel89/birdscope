# Phase portrait

Visualizes the audio signal as a 2D plot using delay coordinates:
each point is `(s[t], s[t+tau])` for a fixed delay `tau`.
Shows dynamical structure that the spectrum does not.

## Decisions for v1

- Method: delay embedding `(s[t], s[t+tau])`.
- tau: fixed at 8 samples (~180 microseconds at 44.1 kHz).
- Drawing: connected line through consecutive points.
- Window size: 1024 samples per repaint.
- Layout: toggle button switches between spectrum and phase portrait;
  only one is visible at a time.
