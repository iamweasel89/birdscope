# Phase portrait

Visualizes the audio signal as a 2D plot using delay coordinates:
each point is `(sample[t], sample[t+tau])` for some delay `tau`.
Shows the dynamical structure of the signal in a way the spectrum does not.

## Scope

Included:
- 2D plot updated during recording.
- Initial fixed delay `tau`; possibly configurable later.

Not included for now:
- 3D phase portraits.
- Multiple delays simultaneously.
- Persistence.

## Source of code

Written from scratch. The `legacy` branch had a phase portrait attempt
on a side branch (not merged into legacy main), but the decision is to
redo it cleanly rather than port that code.
